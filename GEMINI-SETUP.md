# Google Gemini API Setup Guide

## 🆓 Gemini is FREE (with generous limits)!

Unlike OpenAI which requires payment, Gemini offers:
- **10 requests per minute** (FREE)
- **250 requests per day** (FREE)
- **250,000 tokens per minute** (FREE)

Perfect for personal use!

---

## 🔑 Get Your Gemini API Key (FREE)

### Step 1: Go to Google AI Studio
https://aistudio.google.com/app/apikey

### Step 2: Create API Key
1. Click **"Get API key"** or **"Create API key"**
2. Select **"Create API key in new project"** (or use existing project)
3. Copy the API key (starts with `AIza...`)

### Step 3: Add to Vercel
1. Go to https://vercel.com/kylejoshua878-9097s-projects/kaalaman/settings/environment-variables
2. Click **"Add Environment Variable"**
3. **Key:** `GEMINI_API_KEY`
4. **Value:** Your Gemini API key (starts with `AIza...`)
5. **Environments:** Select all (Production, Preview, Development)
6. Click **Save**

### Step 4: Redeploy
```bash
vercel --prod
```
