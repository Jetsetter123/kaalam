### CRITICAL: Add Gemini API Key to Vercel

Your app is deployed but needs the API key to work:

1. **Get FREE Gemini API Key:**
   - Go to: https://aistudio.google.com/app/apikey
   - Click "Create API key"
   - Copy the key (starts with `AIza...`)

2. **Add to Vercel:**
   - Go to: https://vercel.com/kylejoshua878-9097s-projects/kaalaman/settings/environment-variables
   - Click "Add New"
   - Name: `GEMINI_API_KEY`
   - Value: Your API key
   - Environments: Check all boxes
   - Click Save

3. **Redeploy (automatic after adding key):**
   - Vercel will automatically redeploy
   - Or run: `vercel --prod`

## 🆘 Troubleshooting

### "Gemini API key not configured"
- Add `GEMINI_API_KEY` to Vercel environment variables
- Get key from: https://aistudio.google.com/app/apikey

### Still seeing old UI with "AI PROVIDER"?
- Hard refresh: `Ctrl + Shift + R`
- Clear browser cache
- Try incognito mode
- Wait 1-2 minutes for deployment to propagate

### Chat not working?
- Make sure `GEMINI_API_KEY` is added to Vercel
- Check Vercel deployment logs for errors
- Verify the key is valid

### OAuth errors (Google/GitHub login)?
- Update redirect URIs in Google Cloud Console
- Update callback URL in GitHub settings
- See `SETUP-INSTRUCTIONS.md` for details
