Configure Remote API

To use Kaalaman without running it locally, you need to deploy it to a hosting service first.

📋 Setup Instructions

  1. 1. Deploy to Vercel (Recommended):
    • Install Vercel CLI: npm i -g vercel
    • Run: vercel
    • Follow the prompts to deploy
    • Add your OpenAI API key in Vercel dashboard → Settings → Environment Variables
  2. 2. Update Configuration:
    • Open .env.local
    • Set NEXT_PUBLIC_API_URL to your deployed URL
    • Example: NEXT_PUBLIC_API_URL=https://kaalaman.vercel.app
  3. 3. Build Static Version:
    • Run: npm run build
    • Run: npx serve out
    • Or open out/index.html directly in your browser

⚠️ Alternative: Use Lite Mode

If you don't want to deploy, you can use Lite Mode which works entirely in your browser without any server. It provides basic summaries, quizzes, and explanations using local processing. Just toggle "Lite Mode" on in the app interface.

💡 Other Deployment Options

  • Netlify: Connect your GitHub repo and deploy automatically
  • Railway: Deploy with one click from GitHub
  • Render: Free tier available for static sites
  • GitHub Pages: Export static site with next export
